방화벽

NAT_network address translation table
공인 IP를 사용하는 외부 인터넷과 사설IP를 사용하는 내부
네트워크를 분리
내부컴퓨터에서 외부에 접속하기 위해서 NAPT를 사용

패킷필터링_3계층 영역
패킷의 IP주소와 포트 번호를 가지고 일정한 규칙을 
만들어서 통과와 차단기능을 수행
미리 규칙을 규정함(IP나 포트를 정하여 필터링)

프록시 서버_응용계층
네트워크 프로그램의 일종으로
서버와 클라이언트 간에 특정 서비스를 중계해 주는 기능
중계해주는 과정에서 보안처리
URL을 통하여 검사

경계 네트워크_perimeter network
외부 네트워크와 내부 네트워크 사이에 
경계 네트워크(DMZ)를 두어서 분리하는 기능
공개 서버들은 경계 네트워크에 위치시키고
보안이 요구되는 서버들은 내부 네트워크에 두어 분리한다.

배스천 호스트_bastion host
독일어로 성곽 모서리의 둥근 부분을 뜻함
보호하고자 하는 네트워크 외부에 존재하여 보호한 네트워크로 접근
권한을 인증하여 통제

침입탐지시스템
이미 내부 네트워크로 들어온 침입자의 위해 행위를 탐지
데이터 수집 -> 필터링 및 축약 -> 침입탐지 -> 책임 추적성과 대응
네트워크 기반 탐지시스템, 호스트 기반 탐지시스템
